# Car Data Wrangling Project

## Overview
This project focuses on cleaning and preprocessing a car dataset for analysis using Python and Pandas.

## Objectives
- Handle missing values
- Correct data types
- Convert mpg to L/100km
- Normalize data
- Bin horsepower
- Encode categorical variables
- Merge dummy variables
- Save cleaned dataset

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Output
- Cleaned dataset ready for analysis: `data/cleaned_data.csv`
- Histogram plot of horsepower bins
